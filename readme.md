# ViT классификатор

## Задача классификации изображений с использованием Visual Transformer

Набор данных: [Tiny ImageNet](https://www.kaggle.com/datasets/akash2sharma/tiny-imagenet)

Для запуска кода скачайте необходимые библиотеки (если у Вас доступна CUDA):

```
pip install -r requirements.txt
```

Также необходимо иметь датасет Tiny ImageNet в папке data рядом с app.py

Для запуска выполните следующую команду:

```
streamlit run app.py
```
